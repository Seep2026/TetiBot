# Art Pipeline

TetiBot's current art pipeline centers on the Teti Track 128x128 high-resolution master production chain. These assets are committed under `pet-runtime/assets/master_128/` and are the upstream source for later 64x64 production sprites plus 48x48 and 32x32 runtime/test variants.

Start here:

- `aseprite_setup.md` explains how the Aseprite CLI is resolved.
- `aseprite_workflow.md` explains the source files and script boundaries.
- `master_128_pipeline.md` lists the exact output contract.
- `rebuild_on_new_mac.md` is the fresh-machine checklist.
- `asset_naming.md` documents naming rules.
